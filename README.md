# Laboratory work 1

Streaming Twitter Sentiment Analysis System

## General Info

This laboratory work is a system that processes real Twitter API data using SSE's (Server Sent Events) and computes the sentiment scores of the tweets extracted.

## Technologies

The project is created in Erlang with:
- rebar3
- shotgun library
- jcx library
